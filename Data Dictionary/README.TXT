OVERVIEW
--------

The S1000D Data Dictionary provides a listing of all elements and attributes in the S1000D schemas.
Three categories are available. Elements, Attributes, and XML Schemas.
Select the Elements column to get a listing of all elements.
Select the Attributes column to get a list of all attributes.
After selecting the Elements or Attributes column the list is broken by the first letter of the element or attribute name. 
Click the letter that corresponds to the first letter of the element or attribute. 
For example, to locate the copyright element, click on the letter C.
Clicking on copyright will provide additional information about the element including a Graphic view of the element structure,
the definition of the element, the element’s location in the specification, the element’s attributes, etc.

To get a listing of all of the available schema types click XML Schema.
Select a specific schema to see the structure of the schema.Selecting a schema gives the ability to drill down
through a schema and examine the structure of the schema.

NOTE: The structure of the elements shown in the Element Definition is created from the common definition of the element.
Occasionally the structure of an element is redefined in an individual schema. The structure shown in the Graphic view shows
the structure of the element in context and includes any redefinitions. Clicking Master Graphic returns the graphic to the common definition.
Clicking a schema name listed below Master Graphic will show the schema structure as redefined in the schema. See the actual XSD schema for
the final authority and definition of an element. 


HOW TO USE
----------

To access the data dictionary, simply open the file called Index.html in a browser window. Scripting must be enabled.


ERRATA
----------

- Some elements may appear twice on the same page.
- The parents of attributes that are defined as an attribute group are not listed.
- An extra comma appears at the end of lists.
These will be rectified in a future release.
